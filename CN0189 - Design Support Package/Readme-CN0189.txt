Analog Devices Inc.
Design Support Package
CN0189
01/26/2012
Rev. A


This file is designed to provide you with a brief overview of what is contained within the circuit note design support package.


**********************
*******Overview*******
**********************


CN0189 Design Support Package contains: 
	
	Link to Circuit Note
	Link to Product Datasheets
	Schematics
	Bill of Materials	
	Gerber Layout Files
	PADS Layout Files
	Assembly Drawings
	Link to LabVIEW Software for Circuit Note	
	Link to Symbols and Footprints
	Link to Technical Support
	Link to Other Resources



**********************
***File Explanation**
**********************


Circuit Note - Copy of the circuit note this design support package was made for.  This file is provided as a PDF:
	CN0189: http://www.analog.com/CN0189


Product Datasheets - Copy of all ADI component datasheets used in the circuit.  These files are provided as a PDF:
	AD7887:   	http://www.analog.com/AD7887
	AD8608:		http://www.analog.com/AD8608
	ADP3336:	http://www.analog.com/ADP3336
	ADXL203:	http://www.analog.com/ADXL203


Circuit Note Schematic: Provided as a .pdf file and an PADS .sch file:
	EVAL-CN0189-SDPZ-PADSSchematic-RevA.pdf
	EVAL-CN0189-SDPZ-PADSSchematic-RevA.sch


Bill of Material -  A complete list of components used within the circuit.  Details the quantity, value, and manufacturer information.  This file is provided in an Excel format:
	EVAL-CN0189-SDPZ-BOM-RevA.xlsx

Software for CN0189 -- Includes LabVIEW .exe file, DLL, and instructions for running the software in the LabVIEW environment: 
	ftp://ftp.analog.com/pub/cftl/CN0189


Gerber layout files - Layout files provided in Gerber and .pdf format within the .zip file:
	EVAL-CN0189-SDPZ-GRB.zip

PADs layout files:
	EVAL-CN0189-SDPZ-PADSLayout-RevA.pcb
	

Assembly Drawings - Provided as a .pdf file:
	EVAL-CN0189-SCPZ-AssemblyDrawing-RevA.pdf

Symbols and Footprints for circuit board. Provided as an PADS Library file, .pt9:
	EVAL-CN0189-SDPZ-RevA.pt9

System Demonstration Platform:
http://www.analog.com/en/embedded-processing-dsp/blackfin/bf527sdp-hw/processors/product.html

SDP Board User Guide:
http://www.analog.com/static/imported-files/user_guides/SDP_UG_rev1.3.pdf


Symbols and Footprints for ADI parts:

AD7887:
www.analog.com/en/analog-to-digital-converters/ad-converters/ad7887/products/symbols-footprints.html




Technical Support -  If you require further technical assistance please contact us by phone, email, or our EngineerZone community.  http://www.analog.com/en/content/technical_support_page/fca.html
